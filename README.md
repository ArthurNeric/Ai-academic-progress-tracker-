# AI-Based Academic Progress Tracker

A web app for tracking student grades, attendance, and assignments. It flags students who are at risk and gives recommendations.

## Requirements

- Node.js (download from https://nodejs.org)

## How to Set Up

1. Clone the repo:

   git clone https://github.com/ArthurNeric/Ai-academic-progress-tracker-.git

2. Go into the folder:

   cd Ai-academic-progress-tracker-

3. Install the packages:

   npm install

## How to Run

   npm run dev

Then open the link it shows (usually http://localhost:5173) in your browser.

To stop it, press Ctrl + C.
